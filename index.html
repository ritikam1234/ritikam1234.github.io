RAG Speech-to-Speech System

AI Speech Processing · 11-492 Final Project

Developed a modular speech-to-speech (S2S) pipeline integrating Retrieval-Augmented Generation (RAG) to improve domain-specific medical dialogue tasks.

Combined ESPnet ASR, a Haystack-based retriever, and Zephyr-7B LLM with expressive TTS synthesis to address limitations in transcription accuracy and contextual understanding.

Evaluated performance across ASR, LLM, and TTS using WER, BERTScore, UTMOS, and Whisper-based metrics. Used synthetic medical Q&A data and real-time Wikipedia retrieval.

Tools: ESPnet, HuggingFace, Haystack, TTS (kan-bayashi/libritts), gTTS, Google Colab
